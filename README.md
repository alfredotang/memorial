## 🚀 如何啟動專案

---

### 1️⃣ install `nodeJS`
下載 `nodeJS` 👉 [點我到官網](https://nodejs.org/en/)

### 2️⃣ `npm install`
```shell
npm install
```

### 3️⃣ `npm start`
```shell
npm start
```

### 💡 build production code
```shell
npm run build
```
